## Phase -1 第一波执行记录

### 系统体检（12:13）

- Mac arm64（Apple Silicon），236GB 可用
- 系统 Python 3.9.6 太旧，brew 是 Intel 版（pour ruby 失败），改用 uv 官方 curl 安装
- git 2.50.1 可用

### uv + Python 3.10（12:14）

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh   # → ~/.local/bin/uv
uv python install 3.10                             # → 3.10.20
uv venv --python 3.10 .venv
```

### 克隆 Search-R1（12:13）

```bash
mkdir -p external && cd external
git clone --depth 1 https://github.com/PeterGriffinJin/Search-R1.git
```

仓库已加入 `.gitignore`（不污染我们的 git history）。

### 读 README + infer.py + download.py 的关键发现

1. **决策 A 直接落锤**：Search-R1 官方提供预构建 E5 索引下载（HuggingFace 上 `PeterJinGo/wiki-18-e5-index` + `PeterJinGo/wiki-18-corpus`），不需要 BM25 退路。
2. **agent loop 极简**：`infer.py` 60 行完整实现 `<think>/<search>/<information>/<answer>` 循环。
3. **版本鸿沟**：本地 torch 2.11 / transformers 5.6 与 Search-R1 官方 torch 2.4 / transformers <4.48 / vllm 0.6.3 严重不匹配。本地新版只用于读代码和简单 inference，云端必须严格匹配。

### 依赖安装（12:16）

```bash
uv pip install --python .venv/bin/python \
  torch transformers datasets requests huggingface_hub numpy
```

验证：MPS 可用（Apple Silicon GPU 加速），CUDA 自然 False。
