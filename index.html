5 Algorithmic Improvements

5 个算法改进点 · 各自的 paper figure

每张图说明改动的"形状"，配合实证结果验证 / 证伪。

① DFA 5 档 format reward +11% 实证有效

Source: Search-R1 (NQ) · Phase 0d → 0e

改动：把 EM 二元 (0/1) reward 升级为 5 档 DFA：根据 (answer 对错) × (format 合规) × (检索是否命中) 给 0.1 / 0.2 / 0.3 / 0.8 / 1.0 五档分数。

结果：val/nq 从 0.376 → 0.417（+11%）。这是所有 phase 中单次提升最大的一项，远超后续所有 PPO 内部改动。

启示：先把 reward 函数设计对，再调算法。reward 是最大杠杆。

② Sequence-level IS ratio +5% SOTA 决定项

Source: GSPO (Qwen 2025, arXiv:2507.18071) · Phase 2B / 7.8d

改动：PPO ratio 从 token-level exp(log π_new(t) − log π_old(t)) 改为 sequence-level exp(mean_t [log π_new − log π_old])。

结果：DAPO token-level entropy 失控爆到 4.92；GSPO sequence-level 自然探底到 0.46。pg_clipfrac 低 5×。

启示：长 response 下，token-level ratio 方差大；sequence 共享一个 ratio 让多 token 一起 commit，更稳。

③ Multi-turn tool reward shaping -9% reward hacking

Source: ToRL (2503.23383) · Phase 3

改动：把 search 当 first-class tool，每个 search turn 都给 +0.05 召回奖励 + -0.05 重复 query 惩罚（终态 reward 之外）。

结果：response_length 663 → 1937，n_search 1 → 3.5（看似激进 explore），但 raw EM 反降到 0.380。模型学会"刷 shaping bonus"而不是"答对题"。

启示：Single-objective reward 比 multi-component shaping 安全。NQ 单跳本质让 multi-turn 没真正信号。

④ 双层 group advantage (GiGPO) -27% 全 phase 最差

Source: GiGPO (2505.10978) · Phase 4

改动：advantage 分两层。inner = (r − mean_within_prompt) / std（即 GRPO）；outer = (r − mean_across_prompts) / std（跨 prompt 归一化）。total = 0.7 × inner + 0.3 × outer。

结果：困难 prompt 的 mean_across 远高于 inner mean → outer adv 是大负值 → 推 policy 远离 base → entropy 25 步爆到 1.84，40 步用户 kill。最终 val 0.292（-27%）。

启示：cold-start 50 步小窗口下，outer adv 是噪声不是信号。任务-算法不匹配（NQ 单跳 prompt 间相关性弱）。

⑤ Entropy-adaptive controller (ARPO) 部分有效

Source: ARPO (2507.19849) · Phase 5a / 5b

改动：动态 KL β。entropy 超阈（>1.0）→ 自动从 0.001 升到 0.005，超阈消失再降回。

结果：5a 实证 entropy 锁住 0.86-1.0（vs DAPO 4.92），这部分设计成功。但同时叠加的 trajectory credit assignment 让 val 反降到 0.357（-14%）。

启示：ARPO 4 个特性中 adapt entropy ✅ 单测有效，但 traj credit / 局部 KL anchor 都 hurt。Phase 14 计划只继承 adapt entropy 部分，重新组合。
